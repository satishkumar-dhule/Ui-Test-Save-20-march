# QA Team Work Items - Apple Glass Migration

This document enumerates the QA team work items and ownership for the Apple Glass theme migration E2E program.

## Team Roles

- QA Lead (QAL)
- Automation Engineer (AUE)
- Senior QA Engineers (SQA1, SQA2)
- Junior QA (JQA1..JQA6)

## Work Items & Ownership

- QAL: Define E2E test plan, oversee execution, triage defects
- AUE: Set up Playwright, CI integration, test harness, reporting
- SQA1: Create test scripts for agent groups 1-3, review/test new flows
- SQA2: Create test scripts for agent groups 4-6, review/test new flows
- JQA1: Create test scripts for agent groups 7-9, assist with maintenance
- JQA2: Create test scripts for agent group 10 + coordinator, assist with maintenance
- JQA3: Support test data generation and environment setup
- JQA4: Run smoke tests, collect results, maintain suite
- JQA5: Defect logging/tracking, status updates
- JQA6: Documentation, evidence collection, reusability checks

## Status (example)

- Task status is tracked in QA_TRACKER.md and the project todo board.
